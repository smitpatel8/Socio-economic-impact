# Socio-economic-impact
About This Analysis

This project analyzes cancer mortality data from 3,047 U.S. counties, examining relationships between socioeconomic factors and health outcomes. Data sources include the American Community Survey, clinicaltrials.gov, and cancer.gov.

Research Question: How do socioeconomic factors like median household income impact cancer mortality rates in U.S. counties?
